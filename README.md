Introducere:

Proiectul consta in implementarea unui sistem de monitorizare simplu Manager-Agent bazat pe protocolul Simple Network Management Protocol versiunea 1 (SNMPv1). 
Obiectivul principal este de a dezvolta cele doua componente fundamentale (Managerul si Agentul) utilizand strict modulele standard din Python, prin utilizarea exclusiva a modulului socket pentru implementarea si gestionarea stivei de comunicatie.

SNMP este un protocol standard de retea folosit pentru a colecta, modifica si organiza informatii despre echipamentele conectate, permitand administratorilor sa monitorizeze de la distanta performanta si starea retelei.

Acest proiect simuleaza schimbul de mesaje SNMP (de tipul GetRequest si GetResponse) utilizand protocolul UDP, care este standardizat pentru SNMP pe portul 161.
